# Settings UI

This page describes the Settings screen and its sections.

## Sections
- General
- File Paths
- Keyboard
- Performance
- Config System
- Network Updates
- Initial Deployment
- Logging
- Advanced

## Host Integration
- Settings are loaded from the WPF host.
- Save sends updates to the host via WebView2 messages.

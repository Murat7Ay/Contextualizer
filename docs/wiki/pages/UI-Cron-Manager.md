# Cron Manager UI

This page documents the Cron Manager screen in the React UI.

## Features
- List scheduled jobs and scheduler state.
- Filter by status and search by job id or handler name.
- Enable/disable jobs.
- Manually trigger jobs.
- Edit cron expression and timezone.
